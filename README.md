# 即刻取图小程序

#### 介绍
即可取图小程序由极客分享（ https://www.wxshares.com ）开发的一款开源微信/抖音双端小程序，后台用wordpress插件开发，前端用uni-app开发，专为壁纸、头像、表情流量变现开发，遵循GPL V2.0开源协议发布。
